环境启动：
1、分别在./和./nodejs下载开发依赖包 命令 npm install / cnpm install(推荐使用淘宝镜像)
2、到./nodejs目录下，启动node服务器，命令 node app.js
3、在主目录下运行webpack自动打包环境，命令 webapck -wp
4、浏览器输入 localhost:3000 即可
